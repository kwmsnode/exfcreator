# EXFクリエータ
## 概要
EXF形式のビットマップフォントファイルをお持ちのTTFフォントファイルから作成するWebアプリケーション。
## 使用方法
以下のどちらかの方法でご利用ください。
* [こちら](https://kwmsnode.f5.si/serv/exfcreator/)から利用する。（ただし更新が遅れる場合があります。）
* リポジトリをクローンもしくはzipでダウンロード。その後、index.htmlをお使いのブラウザで開く。
