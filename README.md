# EXFクリエータ
## 概要
EXF形式のビットマップフォントファイルをお持ちのTTFフォントファイルから作成するWebアプリケーション。
## 使用方法
リポジトリをクローンもしくはzipでダウンロードします。その後、index.htmlをお使いのブラウザで開いてください。
